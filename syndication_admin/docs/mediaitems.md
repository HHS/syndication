# MEDIA ITEMS

## AUDIO

#### VIEW

From the Syndication Dashboard, click Media menu and Audio submenu.  

![Syndication_Dashboard_Audio.png](https://bitbucket.org/repo/X7nKza/images/2923537894-Syndication_Dashboard_Audio.png)

The Audio List screen will appear.  

![Audio List screen.png](https://bitbucket.org/repo/X7nKza/images/3130562716-Audio%20List%20screen.png)

#### CREATE

5.2.1 From the Audio List screen, click New button, Create Audio screen will appear.

![Create_Audio.png](https://bitbucket.org/repo/X7nKza/images/3738920829-Create_Audio.png)

5.2.2 Enter Name.  Note: there is a 255 character limit.

5.2.3 Optionally, enter Description.  Note: there is a 2000 character limit.

5.2.4 Enter Source URL.  Note: this should begin with http or https.

5.2.5 Optionally, enter Target URL.  Note: this should begin with http or https.

5.2.6 Optionally, enter Custom Thumbnail URL

5.2.7 Optionally, enter Custom Preview URL

5.2.8 Enter Date Syndication Captured [current date will be auto selected].

5.2.9 Enter Date Syndication Updated [current date will be auto selected].

5.2.10 Enter Date Syndication Visible [current date will be auto selected].

5.2.11 Optionally, enter Date Content Authored.

5.2.12 Optionally, enter Date Content Updated.

5.2.13 Optionally, enter Date Content Published.

5.2.14 Optionally, enter Date Content Reviewed.

5.2.15 Select Language from the drop-down menu.

5.2.16 Confirm Active checkbox is checked by default.

5.2.17 Confirm Visible in Storefront/Not Visible in Storefront checkbox.

5.2.18 Optionally, enter the External GUID.  Note: there is a 255 character limit.

5.2.19 Select the Source from the drop-down menu.

5.2.20 Optionally, enter the duration time in seconds.

5.2.21 Select Subscriber from dropdown

5.2.22 Click the Create button.  The Show Audio screen will appear with a confirmation message.  Note: only the completed fields will appear.

![Show_Audio.png](https://bitbucket.org/repo/X7nKza/images/674507285-Show_Audio.png)

5.2.23 Click the List button.  The Audio List screen will appear.

![Audio_List General.png](https://bitbucket.org/repo/X7nKza/images/105359872-Audio_List%20General.png)

#### SORT

5.3.1 From the Audio List screen, click the desired heading (ID, Name, Description, Source URL, Date Syndication Captured, Date Syndication Updated).

5.3.2 Click again.  The order will be changed (descending to ascending).

#### EDIT

5.4.1 From the Audio List screen, click the link desired Name link.  

![Audio_List.png](https://bitbucket.org/repo/X7nKza/images/2470695030-Audio_List.png)

The Show Audio screen will appear.

![Show_Audio.png](https://bitbucket.org/repo/X7nKza/images/2692242557-Show_Audio.png)

5.4.2 Click the Edit button.  The Edit Audio screen will appear.

![Edit_Audio Detail.png](https://bitbucket.org/repo/X7nKza/images/2437518316-Edit_Audio%20Detail.png)

5.4.3 Modify the desired fields.

5.4.4 Either click the Cancel or Update button. The Show Audio screen will appear.  If the Update button was selected then a confirmation message will appear as in the example below.

![Show_Audio updated.png](https://bitbucket.org/repo/X7nKza/images/1140501545-Show_Audio%20updated.png)

5.4.5 Click the List button.  The Audio List screen will appear.
